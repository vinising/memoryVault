# MemoryVault

> ** Your agile-style backlog dashboard, stored as a simple chat.**  

MemoryVault is a high-performance, local-first, zero-install backlog companion for developers, engineers, and technical consultants. Capture ideas, user stories, standard tasks, or bugs instantly in a single portable SQLite database, and analyze it completely offline using a **Progressive Web App (PWA)** combined with local AI models.

---

## 🚀 Key Highlights

1. **Sleek, Polished Dark Theme**: Inspired by premium engineering environments (Linear, GitHub), offering smooth CSS stats card transitions and customizable action triggers.
2. **Zero-Install Local Database**: Built on portable SQLite utilizing **FTS5 Virtual tables** for lightning-fast Full-Text Search. Holds OneDrive/Dropbox friendly multi-instance writes.
3. **Dual Local Reasoning & Fallback**:
   - Queries **Ollama (Gemma 4)** locally at `http://localhost:11434` for secure offline backlog analysis and natural-language query lookup.
   - Automatically re-routes to a high-reasoning **LLM Proxy on Port 8080** if local Ollama times out, is unreachable, or requests specialised delegation.
4. **100% Secure Local Tracing**: Avoids cloud latency and third-party trace tools (like LangSmith or self-hosted Langfuse). It writes call latency, payloads, and model tracking straight to an offline `llm_traces` table.
5. **Mobile-First Progressive Web App (PWA)**: Supports "Add to Home Screen" standard on Android and iOS devices, with custom offline statics caching service workers.
6. **Network Resilience Guarantee**: Captures notes locally in browser `localStorage` if you are on a plane, subway, or server restarts, batch syncing to backend once connection returns.
7. **Extensible Plugin-Architecture**: A simple load-trigger mounts any dynamic `.py` file placed in the `backend/plugins/` directory (includes an autogenerated dynamic CSV export plugin).

## ⚡ Quick Start

### 1️⃣ Prepare Environment
Clone research and navigate to root dir:
```bash
git clone https://github.com/your-org/memoryvault.git
cd memoryvault

# Backend expects python 3.9+ 
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate.bat
pip install -r requirements.txt
```

### 2️⃣ Run Background Launcher
Start the server asynchronously in a visible background terminal using our automated loaders:
- **macOS / Linux**: `./start_server.sh`
- **Windows**: `start_server.bat`

*This boots uvicorn server at `http://127.0.0.1:8000` and pops your browser window immediately.*

## ✍️ Quick Chat Input Console

Enter single-line agile notes directly:
```bash
Bucket | Title | Tags | Description
```

- **Example (Standard Task)**: `TT | Setup API routing middleware | api,infra | Forward queries safely`
- **Example (Epic)**: `EPIC | Release mobile bundle | mobile,app | Build Cordova standalone shell wrapper`

*If you enter plain raw text with no separators, the UI will pop up an elegant selection modal requesting quick bucket classification (EPIC, US, TT, PT).*

## ⚡ Productivity Commands (Slash Commands)

Enter triggers in the console footer:
- **`/todo [Note]`** — Directly save new backlog note.
- **`/done #0004`** — Mark backlog item `#0004` completed.
- **`/status #0004 in-progress`** — Set status directly (`open`, `in-progress`, `done`, `archived`).
- **`/search [Keyword]`** — Search matching records immediately.
- **`/export`** — Trigger JSON backup download.
- **`/import`** — Restore data configuration.
- **`/help`** — Trigger interactive user help menu.

## 🧬 Local AI Configuration

To utilize natural-language summary and chat features offline:
1. Spin up **Ollama** locally.
2. Pull and run model `gemma4`:
   ```bash
   ollama run gemma4
   ```
3. Tap **Summarize Backlog** on your dashboard inside the frontend client.

## 🔌 Dynamic Custom Plugins

MemoryVault automatically checks and mounts plugins from `backend/plugins/` folder on boot.
To register a new API route, simply write a script file exposing the standard register interface:

```python
# backend/plugins/my_custom_plugin.py
from fastapi import FastAPI

def register_plugin(app: FastAPI):
    @app.get("/plugin/hello")
    def hello_world():
        return {"data": "Hello from custom plugins!"}
```
