⚡ Productivity Commands (Slash Commands)

  • /todo Bucket | Title | Tags | Description — Create an entry instantly.
  • /done #0012 — Mark item #0012 completed.
  • /status #0012 in-progress — Set entry state to open, in-progress, done, archived.
  • /search key — Filter search results for key.
  • /import — Trigger JSON data restore.
  • /export — Click download.
  • /help — Raise instruction manuals.
